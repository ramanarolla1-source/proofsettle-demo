## Demonstrated Workflow

```
Verified Sports Intelligence
          │
          ▼
Decision Intelligence
          │
          ▼
Smart Contract Rule Evaluation
          │
          ▼
Decision Artifact
          │
          ▼
Cryptographic Merkle Tree
          │
          ▼
Settlement Authorization
          │
          ▼
Trustless On-Chain Settlement
          │
          ▼
Trust Receipt
```

Every completed decision generates a **Trust Receipt**, enabling users to inspect, understand and independently verify the complete decision journey.
